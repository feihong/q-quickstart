# Cheatsheet

## abs

    abs -23 -1 0 2 55 --> 23 1 0 2 55

## all

    all () --> 1b
    all 78 --> 1b
    all 1 1 1 1 --> 1b
    all 1 1 0 1 --> 0b

## til

    til 0 --> `long$()
    til 8 --> 0 1 2 3 4 5 6 7
